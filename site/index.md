## Operating stance

tappunk builds production-minded local AI infrastructure with a narrow focus: clean interfaces and explicit boundaries.

The goal is not maximal feature count. The goal is dependable behavior.

## What to expect

- stable CLI contracts for humans and automation
- container boundaries you can inspect and reason about
- reproducible environment profiles for agent workflows
- conservative defaults with explicit override paths

If you like systems that stay legible when things fail, this is for you.
