# About tappunk

tappunk is one person operating under a lab identity.

Privacy is intentional. Quality is public.

## Core posture

- privacy-preserving personal identity
- open technical artifacts
- high standards for correctness and operator UX

## Principles

1. **Unix and POSIX first** - clean interfaces, composability, predictable behavior.
2. **Sharp boundaries** - explicit host, container, and runtime contracts.
3. **Production honesty** - clear caveats, diagnostics, and failure modes.
4. **Long-term readability** - code and docs that stay useful as systems scale.

## Audience

Builders and operators shipping local AI systems who value reliability, legibility, and minimal ceremony.
