Toolchain ​

  • gre: parallel git status helper
  • gsty: Ghostty theme TUI tooling
  • utmd: UTM VM lifecycle control

Install via Homebrew ​

bash
brew tap tappunk/tap
brew install tappunk/tap/gre    # parallel git status helper
brew install tappunk/tap/gsty   # Ghostty theme TUI tooling
brew install tappunk/tap/tnk    # local AI orchestrator
brew install tappunk/tap/utmd   # UTM VM lifecycle control
