What we do ​

tappunk ships local AI tooling. Code is public. Interfaces are script-first. Agent workflows run in isolated containers.

What to expect ​

  • CLI interfaces that do not change without notice
  • inspectable container sandboxes
  • environment profiles that reproduce across machines
  • conservative defaults with explicit overrides

Try it now ​

bash
brew tap tappunk/tap
brew install tappunk/tap/tnk
tnk init && tnk run
